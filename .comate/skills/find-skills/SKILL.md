# Skill搜索器

从远端Skills中心搜索并安装新skills。

## 触发条件

**自动触发：**

- 用户提出的问题或任务可能受益于专业skill
- 本地已安装的skills都不适合处理该问题
- 存在远端平台上的某个skill可能更好地处理该请求的可能性

**不触发：**

- 请求简单且不需要专业知识
- 本地已有skill可以处理（本地优先）
- 用户询问与工具使用无关的一般信息

## 安装位置规则

根据用户意图确定skill安装位置：

| 意图            | 安装路径                         | 说明                       |
| --------------- | -------------------------------- | -------------------------- |
| 安装到当前项目  | `.comate/skills/<skill_name>/`   | 仅当前项目可用，随仓库共享 |
| 安装到全局/个人 | `~/.comate/skills/<skill_name>/` | 所有项目可用，个人专属     |

**意图识别关键词：**

- **项目级**：用户提及"当前项目"、"这个仓库"、"项目里"、"团队共享"等
- **全局级**：用户提及"全局"、"个人"、"所有项目"、"通用"等

**无法识别时，主动询问：**

```
请问你希望将技能安装到哪里？

1. **当前项目** (.comate/skills/) - 仅当前项目可用，会随代码仓库共享给团队
2. **全局个人** (~/.comate/skills/) - 所有项目通用，仅自己可用

请选择 1 或 2：
```

安装时若目标目录不存在，自动创建。

## 使用流程

### 步骤1：从Skills中心获取可用Skills

```bash
python3 scripts/fetch_skills.py
```

脚本输出包含skill名称和描述的JSON。失败时输出错误信息并退出。

### 步骤2：分析和推荐

根据获取到的skill描述分析用户请求的相关性，考虑：

- 用户问题中的关键词和领域术语
- 请求的任务类型
- 用户意图与skill描述之间的语义相似度

选择Top 3最相关的skills，不足3个则只展示匹配度高的。

### 步骤3：向用户展示选项

```
发现以下技能可能对你有帮助:

1. **skill-name-1** - skill功能描述
2. **skill-name-2** - skill功能描述
3. **skill-name-3** - skill功能描述

需要我帮你安装哪个技能？（我会先展示详细信息供你确认）
```

### 步骤4：确定安装位置

用户选择skill后，根据上下文判断安装位置：

- 如果用户已明确表达安装位置意图，直接使用对应路径
- 如果无法判断，主动询问用户（参见"安装位置规则"）

### 步骤5：获取下载链接

```bash
python3 scripts/get_download_url.py <skill_name>
```

### 步骤6：预览Skill详细信息

**安装前必须先预览，获得用户确认。**

```bash
python3 scripts/preview_skill.py <skill_name> <download_url>
```

向用户展示（使用Markdown格式，确保每个字段独占一行）：

```markdown
**Skill详细信息：**

- **名称**：<skill_name>
- **描述**：<description>
- **安装位置**：<安装路径>

**功能概览：**

<overview前500字符>

确认安装这个技能吗？（输入"是"继续安装）
```

**等待用户明确确认后，再进行安装。**

### 步骤7：安装Skill

用户确认后，执行安装。根据安装位置传递 `--scope` 参数：

```bash
# 安装到当前项目
python3 scripts/install_skill.py <skill_name> <download_url> --scope project

# 安装到全局个人目录
python3 scripts/install_skill.py <skill_name> <download_url> --scope global
```

### 步骤8：确认安装完成

```
技能 '<skill_name>' 已成功安装!

安装位置: <实际安装路径>

现在可以使用该技能来处理你的请求了。
```

## 错误处理

**API调用失败：**

```
抱歉，暂时没有找到合适的技能来解决这个问题。你可以直接告诉我需要做什么，我会尽力帮助你。
```

**远端没有合适的skill：**

```
Skills中心没有找到特别匹配的技能。我会用现有能力尽力帮你处理这个问题。
```

不要自动重试失败的API调用。
